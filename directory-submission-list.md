# Directory & Backlink Submission List — All 5 Product Lines

Do this in one batch (Day 4), ~2–3 hours total. Every listing below is free to submit.

**Tip:** keep a doc open with product name, tagline, short/long description, logo, and screenshot links — most forms ask for the same 5 fields.

---

## Tier 1 — General SaaS directories (submit all 4 SaaS-style tools)

| Site | Submit URL | Notes |
|---|---|---|
| AlternativeTo | alternativeto.net | List as "alternative to" the closest incumbent |
| SaaSHub | saashub.com | Same alternative-to framing |
| Product Hunt (directory listing) | producthunt.com | Create the page ahead of your scheduled launch |
| BetaList | betalist.com | Good for pre-launch buzz |
| G2 | g2.com | Free vendor profile |
| Capterra | capterra.com | Free basic listing, strong buyer-intent traffic |
| SourceForge | sourceforge.net | Especially relevant for OliFlow |
| Crunchbase | crunchbase.com | Free company + product profile |

## Tier 3 — Line-specific channels

**OliOps Suite** — CRM/business-ops category tags on Tier 1 sites; small-business tool roundup blogs (scan quarterly for new ones accepting free listings)

**OliCommerce Stack** — Shopify Community forums (community.shopify.com); r/shopify, r/ecommerce

**OliFlow Engine** — r/nocode, r/automation; "open source Zapier alternative" roundup articles (email authors requesting inclusion)

**OliConnect** — r/socialmedia, r/marketing, r/smallbusiness; Chrome Web Store / Firefox Add-ons listing itself functions as a discovery directory once published

**Oli-Locator** — different audience entirely, skip Tier 1/2:
- Local Realtor association online directories/forums (free member resource listing)
- BiggerPockets forums (biggerpockets.com)
- r/realtors, r/realestateinvesting
- Direct outreach matters more here than directories — see `outreach-oli-locator.md`
